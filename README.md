# 🛍️ Thrifting App

A full-stack web application where users can create accounts and post items for sale such as yard sale finds, flea market items, or anything worth selling.

This project is built to strengthen my skills in backend development, databases, and building real-world applications.

---

## 🚀 Features

- User authentication (register, login, logout)
- Create, edit, and delete item listings
- Upload images for items
- Each user can manage their own listings
- Persistent data storage using a database
- Responsive UI using Bootstrap

---

## 🧱 Tech Stack

- **Backend:** Python, Flask  
- **Database:** SQLite (planning to use SQLAlchemy)  
- **Frontend:** HTML, Bootstrap  
- **Version Control:** Git & GitHub
